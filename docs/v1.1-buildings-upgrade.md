# LuxCity RP — v1.1 Special Buildings & Ground Upgrade Plan

**Focus:** Improve all 11 special/landmark buildings + sidewalk/ground textures
**Technique:** Canvas-generated textures (CanvasTexture) + additional geometry detail

---

## Ground & Sidewalk Improvements

### Sidewalk Pavement Texture (Canvas-Generated)
**Current:** Flat gray `MeshStandardMaterial({ color: 0xa0a0a0 })` — no detail at all.

**Upgrade:**
- Generate a 512×512 Canvas texture with:
  - Concrete tile grid (1m × 1m squares, thin dark gap lines)
  - Subtle color variation per tile (±5% brightness)
  - Random hairline cracks across some tiles
  - Darker edges near road (dirt/grime buildup)
- Apply as `CanvasTexture` with `RepeatWrapping`
- Generate a matching bump map canvas (grayscale version) for surface depth
- Apply as `bumpMap` on the sidewalk material with `bumpScale: 0.3`

### Road Surface Texture
- Generate asphalt texture with subtle grain/noise
- Apply as CanvasTexture to road material

## Technical Approach

### Canvas Texture Generation Pattern
```
function generateBuildingTexture(width, height, options) {
  const canvas = document.createElement('canvas');
  canvas.width = width; canvas.height = height;
  const ctx = canvas.getContext('2d');

  // 1. Base wall color
  // 2. Window grid (dark rectangles)
  // 3. Floor separators
  // 4. Ground floor detail (doors, storefronts)
  // 5. Surface detail (cracks, stains, patterns)

  const tex = new THREE.CanvasTexture(canvas);
  tex.colorSpace = THREE.SRGBColorSpace;
  return tex;
}
```

### Bump Map Generation
- Create grayscale version of facade texture
- White = raised, Black = recessed
- Apply as `bumpMap` with `bumpScale: 0.2-0.5`
- Gives depth to windows, bricks, tiles without extra geometry
