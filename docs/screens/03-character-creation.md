## Components

- **3D Preview:** `#ccPreview` вҖ” live rotating character model with idle animation
- **Name Input:** `#ccName` вҖ” max 16 characters
- **Name Error:** `#ccErr` вҖ” validation error display
- **Gender Select:** `#ccGender` вҖ” MALE / FEMALE toggle buttons
- **Skin Tone:** `#ccSkin` вҖ” color swatches row
- **Hair Style:** `#ccHair` вҖ” style option buttons
- **Hair Color:** `#ccHairColor` вҖ” color swatches row
- **Top Clothing:** `#ccTop` вҖ” color/style options
- **Bottom Clothing:** `#ccBottom` вҖ” color/style options
- **Backstory:** `#ccBackstory` вҖ” preset backstory options (Hustler, Street Kid, etc.)
- **Enter Button:** `#ccEnter` вҖ” confirms creation and enters game
