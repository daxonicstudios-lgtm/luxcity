## Components

- **Gang Name Input:** `#gcName` — max 20 characters
- **Motto Input:** `#gcMotto` — max 40 characters
- **Color Picker:** `#gcColors` — row of color swatches for gang color
- **Cancel Button:** `#gcCancel` — closes modal
- **Create Button:** `#gcCreate` — creates the gang
