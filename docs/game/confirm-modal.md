## Components

- **Message:** `#cmMsg` вҖ” dynamic confirmation text (default: "Are you sure?")
- **Cancel Button:** `#cmNo` вҖ” dismisses modal
- **Confirm Button:** `#cmYes` вҖ” executes the pending action
- Used for destructive actions (leave faction, disband gang, reset player, etc.)
