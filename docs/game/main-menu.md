## Components

- **Header:** "// MENU" title
- **Close Button:** `#mmClose` вҖ” returns to gameplay
- **Grid:** `#mmGrid` вҖ” grid of menu option tiles
- **Sub-panel:** `#mmSub` вҖ” opens when a menu tile is selected
  - **Back Button:** `#mmBack` (вҶҗ)
  - **Sub Title:** `#mmSubTitle`
  - **Sub Body:** `#mmSubBody` вҖ” dynamic content area

## Menu Items

- MAP вҖ” opens full map view
- STATS вҖ” player statistics
- MISSIONS вҖ” available/active missions
- FACTIONS вҖ” faction panel
- SETTINGS вҖ” game settings (sensitivity, volume, HUD scale)
- ADMIN вҖ” admin/debug panel (if enabled)
