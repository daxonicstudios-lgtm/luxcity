# Emote Wheel

**Element ID:** `#emoteWheel`

## Layout (Mobile Landscape)

```
┌──────────────────────────────────────────────────────┐
│                                                      │
│                    ┌─  WAVE ─┐                       │
│                 DANCE         POINT                   │
│               /                    \                 │
│              TAUNT    (center)   SALUTE              │
│               \                    /                 │
│                 LAUGH         CLAP                    │
│                    └─ SIT ───┘                       │
│                                                      │
└──────────────────────────────────────────────────────┘
```

## Components

- **Wheel Container:** `#emoteWheel` — circular radial menu
- **Emote Options:** arranged in a circle, tap to trigger animation
- Emotes: Wave, Point, Salute, Clap, Sit, Laugh, Taunt, Dance (varies)

## Interaction

- Opens on 😀 button press
- Tap an emote to play the animation
- Tap outside or select to close
